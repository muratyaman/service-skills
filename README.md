# service-skills
basic service for skills

## Requirements
1. Python 3.7.x
2. aiohttp
3. pymongo
4. MongoDB

## Installation
./install.sh

## Configuration
create database and collections
1. mongo
2. use service_skills
3. db.createCollection('profiles')
4. db.createCollection('definitions')

## Execution
./run.sh

HTTP service listens on port 11003
